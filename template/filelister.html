✅ Accepts any Google Drive Folder URL
📂 Fetches all files inside the folder (including nested folders)
📋 Lists file Name, Type, Link, Owner, Last Modified Date, and Parent Folder
📊 Outputs the data in a well-organized table in Google Sheets
⚙️ Runs with a simple button—no coding required!

  • 🤝 Share structured file lists with teams or clients

🔐 Privacy & Security

  • ✅ Runs on your own Google account

  • ✅ Keeps your data secure and private

🧑‍🏫 How to Use It (3-Step Setup)

Step 1: Paste Folder URL

Go to the URL sheet and paste your Google Drive folder link into cell C6.

Step 2: Click “Run Tool”

This activates the built-in script to fetch the file details.

Step 3: View Results

The sheet will automatically switch to the list tab to display the extracted file data instantly.
